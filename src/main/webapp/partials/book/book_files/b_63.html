4. Надежность программного обеспечения

4.1. Общие сведения о программном обеспечении

Программное обеспечение (ПО) представляет собой комплекс информационно связанных программ, тестов, описаний, инструкций пользователя и программиста. С позиции надежности основным ядром ПО являются программы, обеспечивающие решение на ЭВМ различных задач (расчетных, графических, управленческих и т.п.) и/или функционирование АСУ реального времени (АСУ РВ). Далее под ПО понимается только комплекс программ, последовательно выполняемых на однопроцессорной ЭВМ, при этом каждая программа рассматривается как некоторая закодированная информация, записанная на том или ином техническом средстве – носителе (перфокарта, магнитный диск, регистр и т.д.).
