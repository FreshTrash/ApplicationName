Добро пожаловать на наш сайт!

Интернет-ресурс разработан с целью помощи в расчетах показателей надежности автоматизированных систем, а так же для обучения студентов технических вузов по дисциплине "Надежность, эргономика и качество автоматизированных систем". Данный сайт является выпускной бакалаврской работой студента СПБГТИ(ТУ) Фомичева Н.Д. и выпускной магистерской работой студента СГТУ им. Гагарина Ю.А. Каримова М.

При разработке использовались следующие языки и технологии:

  • AngularJS - JavaScript-фреймворк, разрабатываемый Google.
  • Spring - библиотека для упрощения разработки Java программ.
  • Java - объектно-ориентированный язык программирования, разработанный компанией Sun Microsystems.
  • HTML5 - новая открытая платформа, предназначенная для создания веб-приложений использующих аудио, видео, графику, анимацию и многое другое.
  • CSS3 - формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
  • PostgreSQL - свободная объектно-реляционная система управления базами данных (СУБД).
  • GitHub - крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки
  • RESTful API - архитектурный стиль взаимодействия компонентов распределённого приложения в сети.
